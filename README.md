# Data Projects
Technical projects completed during the TripleTen Business Intelligence Analyst Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | Manhattan Airbnb Vacation Rental Market Analysis | Analyzed Manhattan Airbnb listing data in Google Sheets to identify the most attractive neighborhoods and property sizes for vacation rental investment. Evaluated revenue generated by listings to recommend profitable property types based on location and number of bedrooms. |
| 2 | Venture Capital Investment Database Analysis (SQL) | Used SQL to analyze a venture capital database tracking funds, startups, acquisitions, and key industry players. Answered business questions to support a quarterly investment report and provide data‑driven insights for venture capital and startup investors. |
| 3 | E‑commerce Conversion Funnel & Cohort Analysis | Used advanced Google Sheets to analyze raw e‑commerce event logs and turn user activity into business metrics. Built a conversion funnel to evaluate purchase behavior and prepared data for cohort analysis to calculate customer retention rates over time. |
| 4 | Saving SuperStore (Tableau) | Built Tableau visualizations to help a struggling retail superstore improve profitability. Identified key profit centers and loss-makers, recommended products/subcategories to discontinue, evaluated where advertising spend would be worth it, and analyzed return patterns by product/customer to support operational decisions. |
| 5 | SuperStore Returns Analysis Dashboard (Tableau) | Analyzed returned orders to identify root causes of high return rates and created a Tableau dashboard + story for the CEO. Compared return rate vs. total returns to choose the right metric, explored return patterns by product, customer, geography, and time, and proposed actions to reduce returns. |
