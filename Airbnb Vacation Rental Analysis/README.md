# Manhattan Airbnb Vacation Rental Market Analysis

## Overview
This project analyzes Airbnb listings in Manhattan to help a short‑term rental investor identify which neighborhoods and property sizes are most attractive for investment. Using historical listing data, the analysis focuses on understanding demand patterns and revenue potential across locations and bedroom counts.

## Business Question
- Which Manhattan neighborhoods perform best for vacation rentals?
- Which property sizes (number of bedrooms) generate the highest revenue?
- How much revenue do top‑performing listings generate?

## Data
The dataset contains Airbnb listing‑level information, including:
- Neighborhood
- Number of bedrooms
- Pricing and availability
- Revenue metrics

## Tools Used
- **Google Sheets**
  - Pivot tables
  - Advanced formulas
  - Data filtering and aggregation

## Approach
- Aggregated listing data by neighborhood and bedroom count
- Used pivot tables to compare revenue across locations and property sizes
- Identified high‑performing segments based on total revenue generated
- Interpreted results to support investment recommendations

## Key Insights
- Certain Manhattan neighborhoods consistently generate higher rental revenue
- Mid‑sized properties (by bedroom count) tend to balance demand and earnings
- Revenue varies significantly by location, highlighting the importance of neighborhood selection
